# Pasta de Logos de Clientes

Salve aqui os logos das marcas que já confiaram na Barreto's.
Os slots abaixo já estão referenciados no código (`src/lib/constants.ts` → `CLIENT_LOGOS`).

## Arquivos esperados:

| Arquivo | Cliente |
|---------|---------|
| `nestle.png` | NESTLÉ |
| `rio-international-school.png` | Rio International School (RIS) |
| `rjdi.png` | RJDI |
| `lp-glass.png` | LP Glass |
| `colegio-higino.png` | Colégio Higino |
| `portal-natural.png` | Portal Natural |

## Requisitos
- Formato: **PNG com fundo transparente** (ideal) ou **SVG**
- Resolução mínima: **400px** de largura
- Logos horizontais funcionam melhor
- Prefira versão **monocromática escura** — o site aplica `grayscale` por padrão

## Importante
Só use logos de clientes que **autorizaram o uso da marca** no site da Barreto's.
